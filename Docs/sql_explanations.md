\# SQL Query Explanations



\## Customer Segmentation



\* Identified repeat and new customers using GROUP BY and HAVING clause.

\* Customers with more than one order are classified as repeat customers.



\## Sales Analysis



\* Calculated monthly order trends using DATE\_FORMAT.

\* Grouped data by month to analyze sales growth over time.



\## Revenue Analysis



\* Joined order\_items and products tables to calculate revenue by category.

\* Used SUM(price) to find top revenue-generating categories.



\## Cart Abandonment



\* Calculated percentage of canceled orders.

\* Helps understand customer drop-off behavior.
